# 커밋 누락 파일 — 배포 불일치 함정

## 발생 날짜

2026-04-01

## 증상

로컬에서는 수정이 반영됐는데 배포된 사이트에서는 구 버전이 표시됨.

## 원인

`PCTopNav.tsx` 파일을 편집한 직후 다른 작업(추천인 버그, 웰컴 모달)이 이어지면서
해당 파일을 `git add` 없이 커밋했음.
수정 파일이 로컬에만 존재하고 origin에 반영되지 않아 Cloudflare Pages 빌드에 포함되지 않음.

## 해결

누락된 파일을 별도 커밋으로 push → 정상 배포.

## 재발 방지 규칙 (AI 에이전트 필수 준수)

커밋 전 반드시 아래 순서를 따를 것:

1. `git diff --name-only` 실행 — 이번 작업에서 편집한 파일 목록과 대조
2. 편집한 파일이 전부 `git add` 목록에 포함됐는지 확인
3. 누락 파일이 없을 때만 `git commit` 실행

> 복수의 작업을 연속으로 처리할 때 특히 주의.
> 각 작업 완료 시점에 즉시 커밋하거나, 묶어서 커밋할 경우 반드시 전체 diff 검증 후 커밋.
