# 최우선 SSOT 열람 강제 규칙 (Mandatory Initialization)

모든 작업(코드 수정, 디버깅, 파일 분석, 계획 수립 등)을 시작하기 전, 어떠한 행동보다 **가장 먼저 프로젝트 루트 경로에 있는 `CLAUDE.md` 파일을 열람하고 숙지하라.**

`CLAUDE.md`는 100만 MAU 플랫폼 확장을 위해 네가 반드시 지켜야 할 '무관용 전역 행동 규칙'이자, 세부 도메인 지식(`docs/*.md`)으로 향하는 '진입점(Routing Map)'이다.

이 파일을 읽고 현재 작업에 필요한 제약 사항과 추가로 읽어야 할 도메인 문서를 파악하기 전까지는 **절대 어떠한 파일도 임의로 수정하거나 브라우저 제어 등의 툴을 실행하지 마라.**
