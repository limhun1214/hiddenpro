# HiddenPro - Lessons Learned & SSOT Index

이 문서는 프로젝트(100만 MAU 규모) 진행 중 발견한 치명적인 버그, 기술적 함정, 그리고 아키텍처 결정을 기록하는 중앙 인덱스입니다.
AI 에이전트는 작업 시작 전 반드시 이 문서를 숙지하여 과거의 실수를 반복(회귀)하지 않아야 합니다.

## [핵심 원칙 리마인드]

- 프론트엔드 임시 예외 처리(Fallback) 및 우회 로직 전면 금지
- N+1 쿼리 및 불필요한 네트워크 호출 유발 구조 절대 금지
- 테이블 CRUD 작업 시 4가지(SELECT, INSERT, UPDATE, DELETE) RLS 정책 완전성 반드시 확인
